\chapter{Objetivos}

\section*{Objetivo general}

Coordinar dos robots móviles terrestres bajo un esquema \textit{``líder-seguidor"} para el seguimiento de trayectorias predeterminadas en superficies y ambientes controlados.



\section*{Objetivos Particulares}



\begin{enumerate}

	\item Seleccionar y construir dos sistemas de locomoción para los móviles que les permita moverse en superficies y ambientes controlados.

	\item Diseñar e implementar un control de seguimiento de trayectoria para el robot líder.

	\item Diseñar e implementar un control de seguimiento al líder para el robot seguidor.

	\item Establecer e implementar un método de comunicación apropiado para la configuración líder-seguidor.

	\item Diseñar e implementar un algoritmo para la evasión de obstáculos fijos.

	\item Seleccionar e implementar un sistema de sensado en el robot líder, para su propia localización y detección de obstáculos.

	\item Seleccionar e implementar un sistema de sensado en el robot seguidor, para la identificación del líder.

	\item Diseñar e implementar una GUI para el monitoreo del sistema, la selección de una trayectoria a seguir y delimitar el área de trabajo.

\end{enumerate}
